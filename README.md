# SpaceMorph
A chaotic and satisfying web game where you destroy solar systems with debris, cursor, and bombs

## Features
- Randomly generated solar system with varying stats depending on the round, with 12 planets and 4 suns to randomize from

- Satisying sound and visual effects upon destruction of planets

- Intuitive debris physics used to further cascade and demolish solar systems

- Bomb mechanics and cursor-based attraction

## Tech Stack
React 19 • TypeScript • Vite • Tailwind CSS

## How to Play
All playable with mouse, Z for attraction and X for selecting bomb for QOL
